# ChatterX

ChatterX is a real-time anonymous chat application built with Node.js, Express, Socket.IO, and Multer.

## Features
- Anonymous one-to-one chat
- Interest-based matching
- Next stranger flow
- Typing indicator
- Image and short video sharing
- Dark/light theme
- Modern glassmorphism UI

## Tech Stack
- Node.js
- Express.js
- Socket.IO
- Multer
- HTML
- CSS
- JavaScript

## Run Locally
```bash
npm install
node server.js
