About ChatterX

ChatterX is a real-time anonymous chat platform built to connect users instantly with strangers through live one-to-one conversations.

Project Objective

The main objective of ChatterX is to demonstrate real-time communication, anonymous user matching, interest-based pairing, and interactive chat features using modern web technologies.

How It Works

When a user starts a chat, the system adds them to a waiting queue. If another user is available, both users are connected inside a private Socket.IO room. If users enter similar interests, the system gives priority to interest-based matching.

Key Features

  • Anonymous one-to-one real-time chat
  • Interest-based stranger matching
  • Typing indicator
  • Next button to find a new stranger
  • Media sharing after a safety delay
  • Light and dark theme support
  • Report user option
  • Responsive design for desktop and mobile

Technology Stack

HTML CSS JavaScript Node.js Express.js Socket.IO Multer Render GitHub

Future Scope

  • User authentication for optional verified profiles
  • Database storage for better moderation
  • Audio and video chat using WebRTC
  • Advanced report review system
  • Redis-based queue system for better scalability
