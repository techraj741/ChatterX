Help Center

Learn how to use ChatterX, start a chat, use interests, share media, and report unsafe behavior.

Getting Started

  • Open the ChatterX home page.
  • Add interests if you want better matching.
  • Click Start Chat to search for a stranger.
  • Once connected, type your message and press Send.

Interest Matching

Interests help ChatterX connect users with similar topics. For example, if two users add “music” or “coding”, they are more likely to be matched together.

Using the Next Button

Click Next if you want to leave the current chat and search for a new stranger. To avoid accidental skips, ChatterX asks for confirmation before moving to the next chat.

Media Sharing

Photo and video sharing unlocks after 2 minutes of active chat. This delay encourages safer conversations before media sharing.

Report User

If another user behaves inappropriately, open the action menu using the plus button and select Report User. After confirmation, the current chat ends and ChatterX starts searching for a new stranger.

Troubleshooting

  • If no stranger is found, wait for another user to come online.
  • If the chat disconnects, click Next to search again.
  • If media upload does not work, check file size and format.
  • If the page looks outdated, refresh the browser.
